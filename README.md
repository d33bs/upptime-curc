# CU Research Computing Status

This repository uses [Upptime](https://upptime.js.org) to monitor uptime and response times for CU Research Computing resources and partner services.

Live status page: `https://d33bs.github.io/upptime-curc/`

## Monitored resources

- CURC Website
- CURC Docs
- Alpine (CURC page)
- Globus
- ACCESS
- XSEDE redirect site

## How it works

- Checks run on a schedule via GitHub Actions
- Downtime opens/closes incidents as GitHub issues
- Historical uptime and response-time data are committed in this repo
